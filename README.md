# elm-msgpack

MsgPack decoder for Elm 0.18... work in progress...

# Development setup

## Requirements
    * Eralng OTP/Elixir
    * Phoenix

## After git clone
    * cd backend
    * mix deps.get
    
    * cd backend/assets
    * npm install

    * cd frontend
    * elm-package install

## Starting up
    * cd backend
    * iex -S mix phx.server

## Access from browser
    * localhost:4444
